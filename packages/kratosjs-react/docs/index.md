# KratosJs

KratosJs is a powerful admin panel builder for Node.js and React. It provides a fluent API for building comprehensive admin interfaces with forms, tables, resources, and pages.

## Quick Start

To get started with KratosJs, check out the [Getting Started guide](/getting-started).

## Architecture

KratosJs consists of several packages:

- **@maxal_studio/kratosjs**: Core resource management, forms, tables, and pages
- **@maxal_studio/kratosjs-react**: React components for rendering forms, tables, and pages
